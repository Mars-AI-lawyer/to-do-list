### 项目名称

macOS 桌面待办工具（对标滴答清单）

### 项目定位

开发一款 macOS 原生桌面 TodoList 应用，定位为轻量高效待办管理工具，复刻滴答清单主流核心功能，支持常驻桌面/菜单栏，低内存占用，适配 macOS 系统交互逻辑。
技术栈选择：Electron + Vue3/React + 本地数据库，使用文件数据库实现本地数据持久化，无需云端同步（首期只做本地版）。

### 一、核心功能模块（对标滴答清单）

#### 1. 任务基础管理

1. 新增任务：快速输入标题，支持回车快速创建
2. 任务编辑：修改标题、补充多行备注、修改状态
3. 任务操作：删除、置顶、复制任务、批量选中操作
4. 任务状态：未完成 / 已完成，完成任务支持划线置灰，可一键隐藏/显示已完成项

#### 2. 时间与提醒（滴答核心）

1. 为任务设置：截止日期、具体时间、重复规则（每日/每周/每月重复，基础循环即可）
2. 系统级提醒：任务到期前弹窗提醒 + macOS 系统通知中心提醒
3. 支持「无时间」任务，纯随手待办

#### 3. 分类 & 清单体系

1. 自定义清单（分组）：新建、重命名、删除清单，任务可拖拽归属不同清单
2. 内置默认分组：今日、明日、近期、全部、已完成
3. 支持给清单自定义颜色标签，区分不同类型任务

#### 4. 标签功能

支持为任务添加多标签，可按标签筛选任务，对标滴答标签体系。

#### 5. 视图与筛选

1. 多视图切换：清单视图、时间轴视图、标签视图
2. 筛选功能：按时间、状态、标签、清单筛选任务
3. 排序：支持手动拖拽排序、按时间正序/倒序排序

### 二、UI & 交互规范（适配 macOS + 对标滴答）

1. 视觉风格：简约清爽，跟随 macOS 系统风格，**自动适配浅色/深色模式**
2. 窗口：支持窗口大小拖拽、置顶（桌面顶层显示）、最小化
3. 交互逻辑：快捷键支持（新增任务、搜索、切换视图）
4. 搜索功能：全局搜索任务标题/备注/标签，实时检索

### 三、macOS 系统专属能力（桌面插件核心）

1. 菜单栏常驻：程序最小化后缩至 Mac 顶部菜单栏，点击快速唤起主窗口
2. 开机自启：可选配置开机自动运行程序
3. 悬浮小窗口：支持迷你悬浮面板，快速查看今日任务、一键新建待办
4. 权限：仅使用本地存储、系统通知权限，无多余网络请求（首期纯本地）

### 四、数据要求

1. 所有任务、清单、标签数据**本地持久化**，软件重启、电脑重启数据不丢失
2. 数据存储使用本地文件数据库，无需云端同步、无需登录账号

### 五、技术与交付要求

1. 代码结构清晰，关键逻辑添加中文注释，便于后续迭代维护
2. 提供完整可运行源码、本地运行步骤、依赖安装命令
3. 提供 macOS 打包教程，可编译生成 .dmg 安装包，直接在 Mac 安装使用
4. 兼容主流 macOS 版本（12.0 及以上）
5. 优先保证运行流畅、内存占用低

### 额外补充

首期先实现**本地完整版**，暂不做云端同步、多设备同步、日历联动；完成基础功能后，可再迭代扩展。
